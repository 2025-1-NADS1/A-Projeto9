# Documentação do Banco de Dados

## Tabela Quarto

- *id_quarto*: Identificador único do quarto.
- *nome*: Nome do quarto.

## Tabela Sensor

- *id_sensor*: Identificador único do sensor.
- *tipo*: Tipo de sensor (ex: movimento, temperatura).
- *id_quarto*: Referência ao quarto onde o sensor está localizado.
